Backend của hệ thống bán khóa học lập trình.
Công cụ sử dụng:
1.Framework: nestjs.
2.Language: typescript
3.Database: My SQL.
4.Object Storage: Minio.

Các bước để chạy chương trình.
1.Sau khi mở thư mục nhập:
  cd backend

2.Tiếp theo nhập lệnh sau để cài lại các thư viện:
  npm i --ignore-scripts

3.Sau khi cài đặt các thư viện hoàn tất, tại thư mục backend nhập lệnh sau để chạy chương trình.
  npm run start:dev
